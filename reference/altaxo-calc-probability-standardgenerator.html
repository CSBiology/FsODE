StandardGenerator Type

Namespace: Altaxo.Calc.Probability
Assembly: AltaxoCore.Redist.dll
Base Type: Generator

Represents a simple pseudo-random number generator.

The StandardGenerator type internally uses an instance of the Random type to generat pseudo-random numbers.
